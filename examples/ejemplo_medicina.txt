EJEMPLO MÉDICO: DOSIFICACIÓN DE MEDICAMENTOS

Problema:
Un médico debe calcular dosis combinadas:
- Medicamento A: 12.5 ± 0.8 mg
- Medicamento B: 7.3 ± 0.5 mg

Escenarios:
1. Dosis combinada (sinergia): A + B
2. Dosis ajustada (antagonismo): A - B_ajustado

Donde B_ajustado = 3.2 ± 0.3 mg

Datos para el programa:

Escenario 1:
A = 12.5, δA = 0.8
B = 7.3, δB = 0.5

Escenario 2:
A = 12.5, δA = 0.8
B = 3.2, δB = 0.3

Soluciones:
Escenario 1 - Dosis combinada:
Total = 12.5 + 7.3 = 19.8 mg
δ(total) = √(0.8² + 0.5²) = √(0.64 + 0.25) = √0.89 = 0.94 mg
Resultado: 19.8 ± 0.94 mg

Escenario 2 - Dosis ajustada:
Neta = 12.5 - 3.2 = 9.3 mg
δ(neta) = √(0.8² + 0.3²) = √(0.64 + 0.09) = √0.73 = 0.85 mg
Resultado: 9.3 ± 0.85 mg

Implicaciones clínicas:
- Error relativo combinada: 4.7%
- Error relativo ajustada: 9.1%
- Mayor precisión requerida en dosis bajas
